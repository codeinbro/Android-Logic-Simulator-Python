# Android Logic Simulator Python

**Description**  
Android Logic Simulator Python is a **simple logic gate simulator** that can run on **Android using Python / Pydroid**.  
The purpose is to learn **digital logic gates (AND, OR, NOT, etc.)** and understand **Boolean logic** while keeping a **code backup on GitHub**.

---

## Initial Features
- Simple **AND gate simulation**  
- Can be run directly on **Pydroid / Android**  
- Focused on learning basic logic

---

## How to Run
- Open **Pydroid 3** on Android  
- Run the Python program  
- Enter inputs 0 or 1, the output will appear on the screen
